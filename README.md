# Electronics_Sales_-Data
Recommended Structure and Order
1️. Project Title / Headline
Sales Performance Analysis Dashboard | Power BI Data Analytics Project

2️. Short Description / Purpose
This project focuses on analyzing sales performance using an interactive dashboard built in Microsoft Power BI.
The dashboard provides insights into total sales, profit, product performance, and regional distribution to help understand business trends and support data-driven decision making.

3️. Tech Stack
Python – Data cleaning and preprocessing
Microsoft Excel – Data storage and preparation
Microsoft Power BI – Data visualization and dashboard creation

4️. Data Source
Sales dataset containing 10,000 transaction records
Includes fields such as:
Order Date
Product Category
Region
Quantity
Profit
Sales Amount
The dataset was used to simulate a retail sales analysis scenario.

5️. Features / Highlights
Key Performance Indicators (KPIs)
Total Sales: 26.56M
Total Profit: 4.62M
Total Quantity Sold: 26K

Time Series Analysis
Sales trends analyzed by Year, Quarter, Month, and Day

Product Performance Analysis
Waterfall chart showing contribution of different products (Tablet, Laptop, Printer, Monitor, Mobile)

Regional Sales Distribution
Donut chart showing sales distribution across North, South, East, and West regions

Interactive Dashboard
Dynamic visuals enabling quick business insights
